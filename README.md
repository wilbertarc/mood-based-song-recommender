Mood-Based Recommend System
============================================

Step by step to use the app:

1. Download the zip or clone the project
2. Extract the folder
3. Open the folder in visual studio code
4. After opening the folder in visual studio code, make a new terminal
5. On the terminal, write “cd backend-ml”
6. Once you are in the directory of backend-ml, run “python app.py” in the terminal
7. After letting app.py running successfully, now make another terminal to run the front end
8. In the new terminal, run “npm run dev”
9. Then, it will give a local link to access the app, just click it
